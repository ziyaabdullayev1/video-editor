==============================================

📐 ANA LAYOUT SİSTEMİ
---------------------
1. WRAP CONTAINER (Ana Kapsayıcı):
   .wrap {
     max-width: 1200px;
     margin: 28px auto;        /* Üstten 28px, yatayda ortala */
     padding: 0 16px;          /* Yanlarda 16px padding */
   }

2. GRID LAYOUT (Ana Düzen):
   .grid {
     display: grid;
     grid-template-columns: 1fr 340px;  /* Sol kolon esnek, sağ kolon sabit 340px */
     gap: 16px;                         /* Kolonlar arası 16px boşluk */
   }

   /* Responsive breakpoint */
   @media (max-width: 980px) {
     .grid {
       grid-template-columns: 1fr;      /* Tek kolon düzenine geç */
     }
   }

🎯 SİMETRİK LAYOUT'UN SAĞLANMA PRENSİPLERİ
==========================================

1. MERKEZ HIZALAMA:
   - Ana container: margin: 28px auto (yatayda ortala)
   - Max-width: 1200px (büyük ekranlarda sınırla)
   - Padding: 0 16px (kenarlardan eşit boşluk)

2. GRID SİSTEMİ:
   - Sol kolon: 1fr (esnek, kalan alanı kapla)
   - Sağ kolon: 340px (sabit genişlik)
   - Gap: 16px (kolonlar arası eşit boşluk)

3. PANEL SİMETRİSİ:
   - Tüm paneller aynı border-radius: 12px
   - Aynı padding: 12px
   - Aynı border ve background renkleri
   - Aynı box-shadow efektleri

4. ELEMENT HIZALAMA:
   - Flexbox kullanımı: display: flex
   - align-items: center (dikey ortala)
   - justify-content: space-between (yatayda dağıt)
   - gap: 12px (elementler arası eşit boşluk)

5. TRANSFORM KULLANIMI:
   - translate(-50%, -50%) ile tam ortala
   - translateY(-50%) ile dikey ortala
   - translateX(-50%) ile yatay ortala

6. CONSISTENT SPACING:
   - 4px, 6px, 8px, 10px, 12px, 16px (4'ün katları)
   - Fibonacci benzeri artış: 6px → 8px → 12px → 16px

7. RESPONSIVE TASARIM:
   - 980px breakpoint'te tek kolonlu düzen
   - Flex-wrap ile taşan elementleri alt satıra geçir
   - Min-width değerleri ile minimum boyutları koru

Bu parametreler sayesinde tüm elementler matematiksel olarak simetrik ve hizalı görünür!
